Zombie Survival

Zombie Survival is a top-down 2D mobile game. It was my first project undertaken in Unity and is compatible with both android and iOS devices. The scripting is written entirely in C#. The game implements memory-efficient infinite runner-style room and entity generation, accelerometer and virtual joystick controls as well as 2D lighting and collision detection. The aim of the game is simple: to survive as long as you can while killing as many zombies as you can.


OSRSLookup.com

OSRSLookup is a fansite for the MMORPG 'Oldschool Runescape' (found here). The site allows users to lookup weapon/armour items and compare them against each other in order to view which item has the better stats/is more appropriate for their situation. I implemented the site from scratch (using no web design tools/templates such as wordpress) using a HTML, CSS 3.0, Javascript & JQuery (with AJAX) front end, along with a PHP and MySQL backend. The project and it's aims/ideas were largely influenced by the Runescape community and as such I am still updating this project over time and am hoping to expand its uses to the community's requests.

Gamers

'Gamers' is a project I'm currently working on (and as such, the name 'Gamers' is not confirmed at the moment) which I will be working on over the coming year. It is a social app, currently only aimed towards android, which will connect people by the games they play and their proximity to each other. The app will be coded 99% in Java although I expect to be implementing some SQL.
